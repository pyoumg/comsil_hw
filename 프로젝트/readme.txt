*실행시 모든 키는 한번만 눌러주세요

초기화면:
q- 종료
r-랭킹
스페이스바-게임 설명

게임설명:
스페이스바- 게임시작

게임은 레벨 1,2,3으로 나누어져 있습니다.

게임은 도착지점에 도달or 시간 종료시 종료됩니다.
방향키로 움직일 수 있으며, h를 누르면 정답경로가 1초간 출력됩니다.
시간제한이 있으며, 남은 시간이 출력됩니다.
target에 도달하면 그 이후에는 target이 출력되지 않습니다.

레벨 1,2 플레이 후 결과화면 출력후 스페이스 바: 다음 레벨로 넘어감

레벨 3 플레이 후 결과화면 출력 후 스페이스 바: 이름 입력하라는 메시지가 콘솔창에 뜹니다. 글자 제한은 49자입니다.
*레벨3 플레이 종료시 totalscore가 콘솔창에 표시됩니다.

게임 결과화면은 : 
	target에 도달& 목표지점에 도착
	target에 도달x&목표지점에 도착
	목표지점에 도착x
의 3가지로 나누어져 있습니다.

랭킹시스템:
q: 초기화면 이동
데이터가 아무것도 없을때는 no data가 표시되고, q만 작동합니다.
랭킹시스템에서 아무것도 누르지 않았을 때는 (기본값) 데이터들이 내림차순으로 정렬됩니다.
순위,이름,점수가 표시되며 동점인 경우에는 나중에 들어온 데이터가 더 높은 순위입니다.

1:범위 검색
콘솔창에 x,y를 입력하라는 메시지가 뜨며, 반드시 2개 다 입력해야합니다.
x<=y,y>0,x<=(총 데이터)를 모두 만족하는 경우에만 범위검색이 가능하며, 그렇지 않으면 range error이 표시됩니다.
해당 범위에 있는 데이터가  내림차순으로 정렬됩니다.

2: 이름 검색
이름을 입력하라는 메시지가 콘솔창에 뜨며, 글자제한은 49자입니다.
해당하는 이름이 있으면 해당데이터가 내림차순으로 표시됩니다.
없으면, search failure이 표시됩니다.

3. 삭제
해당되는 랭크값을 콘솔창에 입력합니다.
잘못된 랭크값을 입력하면 delete failure이 출력되고, 그렇지 않으면 delete success가 콘솔창에 출력되고 
삭제 후 데이터 값들을 내림차순으로 표시합니다.
